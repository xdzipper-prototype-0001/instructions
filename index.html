XD.BY- Instructions Manual

Step 1: Make a Github account. If you already have a Github account, then use that.


Step 2: Go to https://github.com/xdBY-domains/register/fork, scroll down all the way, and click "Create fork". Make sure that the branch is main or master.


Step 3: In your Github fork, go into the "copy-paste" folder, and select "configuration.txt", then copy all the contents of configuration.txt into your clipboard.


Step 4: Exit configuration.txt, and the "copy-paste" folder, then click "Add file", then "Create new file", and name the file "your-domain-name.json". Paste in the contents of configuration.txt, customize it, then click "Commit changes", and go back to your repository link.


Step 5: Select "Pull requests", then click "New pull request". After that, click "Create pull request". Customize the title and description, then click "Create pull request".


Your domain will be approved when we check out your website entirely, and checking for abuse. We will email you when domain verification is succeeded.
